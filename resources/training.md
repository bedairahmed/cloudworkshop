## Official Free Training Platforms

### Microsoft Learn (Free)

Microsoft's official learning platform with self-paced modules, learning paths, and free sandbox environments. Content is directly aligned with certification exams. Covers Azure, Microsoft 365, Dynamics 365, Power Platform, and more. Available in 17+ languages.

| Resource | Link |
|----------|------|
| Microsoft Learn Home | https://learn.microsoft.com/en-us/training/ |
| Azure Training Paths | https://learn.microsoft.com/en-us/training/azure/ |
| AZ-900 Learning Path | https://learn.microsoft.com/en-us/training/courses/az-900t00 |
| All Certifications | https://learn.microsoft.com/en-us/credentials/certifications/ |
| Azure Free Account ($200 credit) | https://azure.microsoft.com/en-us/free/ |
| Microsoft Virtual Training Days (free events + exam vouchers) | https://events.microsoft.com/en-us/mvtd |

### Google Cloud Skills Boost (Free)

Google's official hands-on training platform. Offers interactive labs running on real Google Cloud infrastructure, learning paths organized by role, and skill badges you can share on LinkedIn.

| Resource | Link |
|----------|------|
| Google Cloud Skills Boost | https://www.cloudskillsboost.google/ |
| Google Cloud Training | https://cloud.google.com/learn/training |
| Google Cloud Certifications | https://cloud.google.com/learn/certification |
| Google Codelabs (Free step-by-step labs) | https://codelabs.developers.google.com/cloud |

### Linux Foundation Training (Free + Paid)

The Linux Foundation offers both free introductory courses and paid professional certifications. Their Kubernetes certifications (CKA, CKAD, CKS) are hands-on, performance-based exams — widely regarded as the gold standard for container orchestration skills.

| Resource | Link |
|----------|------|
| All Courses & Certifications | https://training.linuxfoundation.org/certification-catalog/ |
| Intro to Kubernetes (Free) | https://training.linuxfoundation.org/training/introduction-to-kubernetes/ |
| Kubernetes Fundamentals (LFS258) | https://training.linuxfoundation.org/training/kubernetes-fundamentals/ |
| Certified Kubernetes Administrator (CKA) | https://training.linuxfoundation.org/certification/certified-kubernetes-administrator-cka/ |
| Certified Kubernetes App Developer (CKAD) | https://training.linuxfoundation.org/certification/certified-kubernetes-application-developer-ckad/ |
| Kubernetes & Cloud Native Associate (KCNA) | https://training.linuxfoundation.org/certification/kubernetes-cloud-native-associate/ |
| CNCF Certifications Overview | https://www.cncf.io/training/certification/ |

### Docker Training (Free + Paid)

Docker provides official training content through partnerships with LinkedIn Learning and Udemy. The Docker Certified Associate (DCA) exam is administered by Mirantis and validates real-world Docker skills for professionals with 6–12 months of experience.

| Resource | Link |
|----------|------|
| Docker Official Training | https://www.docker.com/trainings/ |
| Docker Docs & Tutorials | https://docs.docker.com/get-started/ |
| Docker Certified Associate (DCA) Exam | https://training.mirantis.com/certification/dca-certification-exam/ |

## Paid Training Platforms

### Pluralsight

Subscription-based platform with thousands of video courses and hands-on labs across cloud, DevOps, security, and software development. Includes skill assessments and role-based learning paths. Offers a free trial.

Link: https://www.pluralsight.com/

### Udemy — Scott Duffy (AZ-900)

Scott Duffy is a Microsoft MVP and TOGAF-certified architect with 1M+ students. His AZ-900 course is the most popular Azure Fundamentals exam prep on Udemy — always updated with the latest exam requirements. Includes practice tests and a study guide.

Link: https://www.udemy.com/course/az900-azure/

## Recommended Learning Path

1. **Cloud Concepts** — Watch the Cloud Basics videos above to understand core ideas
2. **Azure Fundamentals** — Complete the AZ-900 path on Microsoft Learn (free)
3. **Get Certified** — Take the AZ-900 exam ($99) to validate your knowledge
4. **Hands-On Labs** — Practice on Google Codelabs and Cloud Skills Boost (free)
5. **Containers** — Docker Tutorial → Kubernetes Course (YouTube)
6. **Infrastructure as Code** — Terraform Full Course (YouTube)
7. **DevOps Career** — DevOps Bootcamp playlist + consider AZ-104 or AZ-204 next
